# 维护者学习路径

这条路径面向需要维护仓库结构、校验脚本和教学口径的人。

## 目标

- 知道哪些约定已经被 lint 化
- 知道教学目录和案例目录如何一起演进
- 知道新增 skill 时应该同步补哪些索引和检查

- README / docs / teaching 的入口是否闭合
- 新 skill 是否补了 docs、checker、eval case
- lint、docs checker、prototype checker 是否一起更新
