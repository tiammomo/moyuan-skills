# Skill 作者学习路径

这条路径面向已经准备自己写或重构 skill 的作者。

## 目标

- 把需求收窄成可触发 skill
- 设计渐进式披露结构
- 给 skill 补上 checker、样例和文档入口

2. [04-progressive-disclosure-workshop.md](./04-progressive-disclosure-workshop.md)
3. [07-case-gradient.md](./07-case-gradient.md)

## 写 skill 时重点看什么

- `Task Router` 是否足够窄
- `Progressive Loading` 是否说清楚了何时读哪个资源
- `Safety First` 是否真的覆盖高风险路径
- checker 是否真的能跑通主链路
