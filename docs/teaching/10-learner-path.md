# 初学者学习路径

这条路径面向第一次接触 skill 和 harness thinking 的读者。

## 目标

学会三件事：

- 看懂这个仓库每一层在干什么
- 自己做出一个最小可用 skill
- 知道 skill 为什么不能写成一篇越来越长的总说明书

## 建议顺序

1. [01-learning-map.md](./01-learning-map.md)
2. [02-read-the-repo.md](./02-read-the-repo.md)
3. [03-build-your-first-skill.md](./03-build-your-first-skill.md)
4. [04-progressive-disclosure-workshop.md](./04-progressive-disclosure-workshop.md)

## 学完后应该能做到

- 分清 frontmatter、`SKILL.md`、`references/`、`scripts/`、`assets/`
- 看懂一个 router 是怎么把问题收窄的
- 做出一份带最小 checker 的 skill 草稿
