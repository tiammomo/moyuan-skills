# Review Checklist

## Contents

- [Before Circulation](#before-circulation)
- [Before Rollout Approval](#before-rollout-approval)

## Before Circulation

- 风险等级是否和 diff 一致
- breaking changes 是否写清楚受影响面
- migration notes 是否可执行
- open questions 是否显式列出

## Before Rollout Approval

- owning team 是否确认了 rollout checks
- contract test / compatibility test 是否已经补齐
- 需要的人类审批是否已完成
- 是否还有未确认的客户端依赖
