# Future Roadmap

## Stage 1: Static Skill

Start with a clear trigger and a stable router.

This is the smallest reusable unit.

## Stage 2: Progressive Skill

Split topic depth, scripts, and assets outward so the skill stays navigable.

This makes the bundle easier to maintain and easier to compose.

## Stage 3: Skill With Local Checks

Add smoke checks and representative replay tasks.

At this stage, the skill begins to support repeatable quality control.

## Stage 4: Skill With Harness Support

Add system-level contracts around:

- tool execution
- memory or state
- review gates
- eval pipelines

This is where the skill becomes one layer in a larger agent system.

## Stage 5: Skill Ecosystem

Coordinate multiple skills through shared harness rules.

At this point, design quality depends less on any one skill and more on how the whole system routes, measures, and governs work.
